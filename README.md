# ✈️ Fighter Jet Shooter Game

A simple **2D Fighter Jet Shooter Game** developed using **C# Windows Forms** as a Human-Computer Interaction (HCI) semester project. The game interface was first designed in **Adobe Illustrator** and then implemented using **C#** with event-driven programming concepts in Visual Studio.

---

## 📌 Project Overview

The Fighter Jet Shooter Game is a desktop application where the player controls a fighter jet to destroy enemy aircraft while avoiding them from reaching the bottom of the screen.

This project demonstrates the application of **Human-Computer Interaction (HCI)** principles, graphical user interface design, and object-oriented programming using Windows Forms.

---

## 🎯 Objectives

- Design an interactive and user-friendly game interface.
- Apply Human-Computer Interaction (HCI) principles.
- Implement event-driven programming using C#.
- Develop a desktop game using Windows Forms.
- Improve programming and graphical design skills.

---

## 🛠 Technologies Used

- **Programming Language:** C#
- **Framework:** Windows Forms (.NET Framework)
- **IDE:** Microsoft Visual Studio
- **Graphics Design:** Adobe Illustrator
- **Version Control:** Git & GitHub

---

## 🎮 Features

- ✈️ Player-controlled fighter jet
- 🎯 Bullet shooting system
- 💥 Collision detection
- 👾 Three enemy fighter jets
- ☁️ Animated moving clouds
- 📈 Dynamic difficulty (enemy speed increases with score)
- 🏆 Live score tracking
- 🔄 Restart game functionality
- ❌ Game Over screen

---

## ⌨️ Controls

| Key | Action |
|------|--------|
| ⬅ Left Arrow | Move Left |
| ➡ Right Arrow | Move Right |
| Space Bar | Shoot Bullet |
| Enter | Restart Game |
